A \textit{symmetric bilinear form} is a map \(k : \X \times \X \to \RR\) over a vector space \(\X\) such that, for all \(x,y,z \in \X\), \(\alpha, \beta \in \RR\),
\begin{enumerate}
    \item \(k(x,y) = k(y,x)\) and
    \hfill (symmetry)
    \item \(k(\alpha x + \beta y, z) = \alpha k(x,z) + \beta k(y,z)\).
    \hfill (bilinear)
\end{enumerate}
